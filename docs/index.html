📐 주요 지표 설명

판매량(추정)
SteamSpy owners 범위(예: 200,000 ~ 500,000)의 중간값으로 환산. 번들·증정·무료배포 포함될 수 있어 실제 유료 판매량과 차이 있음
동접자
Steam 공식 API의 실시간 동시접속자. 수집 시점(20:00 KST) 스냅샷이며, 매일 같은 시각 비교 시 의미 있는 추세 파악 가능
전일증감
전날 동일 수집 스크립트 실행 시점의 스냅샷과 비교. 수집 시각이 크게 다른 경우 오차 발생 가능
순위 변동
전날 Steam CCU 기준 순위 대비 오늘 순위 변화. ▲ 상승 / ▼ 하락 표시
리뷰(긍정%)
SteamSpy 기준 전체 리뷰 수 및 긍정 비율. ≥95% 매우 긍정 ≥75% 긍정 <75% 복합

🎯 퍼블리싱 후보 — 선별 기준

① 순위
Steam CCU 기준 동시접속자 순위 1,000위 이내 — 매일 수집 시점 기준
② 개발사
퍼블리셔·개발사 명칭에 대형사 키워드 미포함. 퍼블리셔 미기재 시 개발사 명칭 적용
③ 출시 경과
기본값 전체. 필터바 "출시" 드롭다운으로 1·2·3·5년 이내 선택 적용 가능. 출시일 미기재 게임은 항상 통과
④ 판매량
기본값 전체. 필터바 "판매량" 드롭다운으로 50만·100만·200만·500만 미만 선택 적용 가능. 판매량 미기재 게임은 항상 통과

Valve · EA · Ubisoft · Activision · Blizzard · 2K · Bethesda · Square Enix · Bandai Namco · Capcom · SEGA · Nexon · Netmarble · NCSoft · Krafton · Smilegate · Riot · Epic · Microsoft · Sony · Nintendo · CD Projekt · Paradox · THQ Nordic 등

표시 범위

데이터 소스

탭 구조
월별 탭(1단계) → 클릭 시 해당 월 일별 탭(2단계) 펼침. 오늘 날짜는 금색 ★로 표시
★ 인기
Steam popularcomingsoon 순위 기준 — 일별 뷰 상위 5개 / 월별·전체 뷰 상위 10개. 카드 앞부분에 정렬되며 금색 테두리로 구분. Steam 자체 알고리즘(검색·위시리스트·팔로워 등 종합)으로 산출된 인기도 순위
날짜 미정
출시일이 "To Be Announced" 또는 연·월만 확정된 경우 미정 탭에 표시
Steam 공식 API 응답 기준이므로 실제 출시일과 다를 수 있음. popularcomingsoon 순위는 Steam 알고리즘에 따라 매일 달라질 수 있음
